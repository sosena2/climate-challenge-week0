# Notebooks

This folder stores the country-specific EDA notebooks used for the climate challenge.

Each branch should keep its own notebook here, alongside the branch-specific raw CSV used by that notebook.

## Branch Convention

- This branch is Kenya-only.
- The notebook name should be `kenya_eda.ipynb`.
- The notebook README and root README should describe the Kenya CSV and analysis.
- Do not commit the `data/` folder contents; those files stay local.
